# Xola dos Santos Resumé

> Xola dos Santos Resumé, built using Next.js. The app which also checks the current Bitcoin value in USD, GBP and EUR

## Quick Start

``` bash
# Install dependencies
npm install

# Serve on localhost:3000
npm run dev

# Build for production
npm run build
```
